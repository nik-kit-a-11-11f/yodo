# Урок: Изменение текстового содержимого с помощью JavaScript

## Задача от кофейни "Утренний Бриз"
Кофейня "Утренний Бриз" ежедневно обновляет акции. На странице УЖЕ есть заголовок с текстом "Добро пожаловать в нашу кофейню!". Ваша задача - написать код, который автоматически изменит его на акционное предложение.

## Задание
Напишите JavaScript код, который:
1. Найдет существующий заголовок `<h1>` с помощью `document.querySelector`
2. Изменит его текст на "Акция дня: Скидка 20% на латте!" используя свойство `.textContent`

## Требования
- Используйте правильный селектор для поиска элемента `<h1>`
- Примените свойство `.textContent` для изменения текста
- Код должен работать после загрузки страницы

## Проверка
Откройте HTML-файл в браузере и убедитесь, что заголовок изменился с приветственного на акционный.
